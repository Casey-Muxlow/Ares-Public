ARES SYSTEM

Analytics, Reporting, and Evaluation System.

ARES Analytical System

ARES will analyize the combat effectiveness of a user based upon the stats for the game and platform.

ARES Reporting System

ARES will report to the user the stats they wish to be shown for the platform and game that they wish to have gathered.

ARES Evaluation System

ARES evaluates numerous aspects of a users data from the users K/D to overall kills. ARES will then compare users located within the same guild and give an overall evaluation of all members of the guild.

ARES Build Evaluation System

This is the tool that sets ARES apart from basic stat bots. ARES does not just provide stats for numerous games. ARES features links to the most popular build tools for games such as Division 2, COD, Battlefield and if the tools come available even ARC Raider.
