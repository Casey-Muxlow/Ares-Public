ARES SYSTEM

Analytics, Reporting, and Evaluation System.

ARES Analytical System

ARES analyzes the combat effectiveness of a user based upon live telemetry from the game and platform.

ARES Reporting System

ARES retrieves and reports specific gameplay metrics, tailored to the platform and game title requested by the operator.

ARES Evaluation System

ARES evaluates numerous aspects of a user's data, from combat ratios (K/D) to objective efficiency. It provides comparative analytics for members within the same guild to establish a performance hierarchy.

ARES Build Evaluation System

The definitive ARES feature. Beyond raw stats, ARES cross-references equipped gear with meta-build repositories for titles like The Division 2, COD, Battlefield, and ARC Raiders.
